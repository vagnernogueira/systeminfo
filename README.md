# SystemInfo
Um pequeno projeto web/JSP para testes que exibe informações do sistema.

### Uses

  - http://localhost:8080/systeminfo/

  - http://localhost:8080/systeminfo/network.jsp

  - http://localhost:8080/systeminfo/jboss.jsp
